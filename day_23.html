Phase 23: Reading the Ledger

Professor's Briefing

A true master of data doesn't manually type out their datasets. They read them from files. The most common file format for this is the CSV (Comma-Separated Values) file. It's just a simple text file that looks like a spreadsheet. Today, you'll learn to use Pandas to read a CSV directly into a DataFrame.

The Blueprint: Reading a CSV with Pandas

Pandas makes this incredibly easy with one function: `pd.read_csv()`. You give it the path to the file, and it does all the work of parsing it into a clean DataFrame.

Code Workbench

Imagine we have a text file named `candidates_sheet.csv` with this content:

name,email,status
Priya Sharma,priya@email.com,Interviewed
David Chen,david@email.com,Hired
Aisha Khan,aisha@email.com,Offer Sent

Now, our Python code to read and analyze it.


import pandas as pd

# Name of the file containing our candidate ledger.
filename = 'candidates_sheet.csv'

try:
    # Use the read_csv function to load the ledger into a DataFrame.
    df = pd.read_csv(filename)

    print("--- Successfully loaded candidate ledger ---")
    print(df)

    # Now we can analyze it. How many candidates per status?
    # The .value_counts() method is perfect for this.
    print("\n--- Breakdown by Status ---")
    status_counts = df['status'].value_counts()
    print(status_counts)

except FileNotFoundError:
    print(f"ERROR: Ledger file '{filename}' not found. Cannot proceed.")
