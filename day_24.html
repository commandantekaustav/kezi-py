Phase 24: Slicing the Intel

Professor's Briefing

We rarely need the entire ledger. Usually, we need a specific slice: "Show me all the hired candidates," or "Show me the candidates with a score over 90." In Pandas, this is called **filtering** or **boolean indexing**. You ask a `True`/`False` question, and the DataFrame shows you only the rows where the answer is `True`.

The Blueprint: Filtering a DataFrame

The syntax is powerful. `df[df['column'] == 'value']`. It looks strange, but it's a logical sequence: inside the brackets, you define the condition (`df['column'] == 'value'`), which produces `True` or `False` for each row. The outer brackets then use that result to show only the `True` rows.

Code Workbench


import pandas as pd

# Let's use our crew roster again.
crew_data = [
    { "name": "Priya Sharma", "role": "Data Scientist", "score": 92 },
    { "name": "David Chen", "role": "ML Engineer", "score": 88 },
    { "name": "Kezia Marian", "role": "Lead Recruiter", "score": 100 }
]
df = pd.DataFrame(crew_data)
print("--- Full Roster ---")
print(df)

# Condition: Show me only members with a score greater than 90.
high_performers = df[df['score'] > 90]
print("\n--- High Performers (Score > 90) ---")
print(high_performers)

# Condition: Show me the Lead Recruiter.
the_recruiter = df[df['role'] == 'Lead Recruiter']
print("\n--- The Lead Recruiter ---")
print(the_recruiter)
