Phase 22: The Master Spreadsheet

Professor's Briefing

You have met this specialist before. The ultimate data organizer. Now you will learn to use it. **Pandas** is the most essential library for data analysis in Python. Its core tool is the **DataFrame**—a powerful, in-memory version of an Excel spreadsheet. Today, you'll learn to create one.

The Blueprint: The Pandas DataFrame

You can create a DataFrame from a list of dictionaries—the exact structure you learned in Phase 18. Each dictionary becomes a row, and each key becomes a column header.

Code Workbench


# First, we must import our specialist
import pandas as pd # The 'as pd' is a common alias, a shorter codename.

# This is our crew roster from Phase 18
crew_data = [
    { "name": "Priya Sharma", "role": "Data Scientist", "score": 92 },
    { "name": "David Chen", "role": "ML Engineer", "score": 88 },
    { "name": "Kezia Marian", "role": "Lead Recruiter", "score": 100 }
]

# Create the DataFrame from our list of dictionaries
df = pd.DataFrame(crew_data)

# The print() function now displays a beautifully formatted table.
print("--- Crew Roster DataFrame ---")
print(df)

# You can get basic stats with the .describe() method
print("\n--- Mission Score Analysis ---")
print(df['score'].describe())

Why This Matters To You

This is it. The bridge between your first course and this one. Almost every 'Historian' (Data Analyst) and 'Oracle' (Data Scientist) you interview lives and breathes in Pandas DataFrames. It is their primary workbench. Understanding a DataFrame is understanding their world.
