Phase 25: Presenting the Blueprint

Professor's Briefing

Raw data is for analysts. Leadership needs a clean presentation. A visual. The `Matplotlib` library is Python's foundational tool for creating charts and graphs. Today we bring in this new specialist to create a simple bar chart from our data, to present our findings with impact.

The Blueprint: Charting with Matplotlib

After using Pandas to get our data summary, we can pass that summary to Matplotlib to create a visual plot. It's a two-step process: analyze, then visualize.

Code Workbench


import pandas as pd
# We import the 'pyplot' part of matplotlib, and give it the alias 'plt'.
import matplotlib.pyplot as plt

# Let's use our ledger file from a few days ago.
# candidates_sheet.csv:
# name,email,status
# Priya Sharma,priya@email.com,Interviewed
# David Chen,david@email.com,Hired
# Aisha Khan,aisha@email.com,Offer Sent
# Alex Ray,alex@email.com,Hired

try:
    df = pd.read_csv('candidates_sheet.csv')

    # 1. ANALYZE: Get the counts of each status.
    status_counts = df['status'].value_counts()

    # 2. VISUALIZE: Create the plot.
    status_counts.plot(kind='bar')

    # Add labels and a title to make the chart professional.
    plt.title('Candidate Pipeline Status')
    plt.xlabel('Status')
    plt.ylabel('Number of Candidates')
    plt.xticks(rotation=0) # Keeps the x-axis labels horizontal
    
    # Display the chart.
    plt.show()

except FileNotFoundError:
    print("ERROR: Cannot find ledger file.")

Professor's Note: You see how the story comes together? Data to table, table to numbers, numbers to a visual story. It is a beautiful process. Much like watching you learn.
