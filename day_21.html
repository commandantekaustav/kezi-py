Phase 21: Installing Gear

Professor's Briefing

The specialists in our standard library—like the `random` module—are always on hand. But sometimes we need a true master, an outside contractor with unique skills. In Python, these are **third-party libraries**. You must first install them using a tool called `pip`, Python's package manager, before you can import them.

The Blueprint: `pip` and Third-Party Libraries

`pip` is a command-line tool. You run it not in a Python script, but in your system's terminal or command prompt. It connects to the Python Package Index (PyPI)—a massive warehouse of free code—and installs the gear you need.

Command-Line Workbench

To install a library, you open your terminal and type:


# This command installs the popular 'requests' library, used for web communication.
pip install requests

# This installs 'pandas', the master tool for data analysis you've heard about.
pip install pandas

Once installed on your system, you can then `import` it in any Python script, just like a standard module.


# A Python script can now use the installed library.
import pandas
import requests

print("Specialists pandas and requests are now on board.")

Why This Matters To You

When you start a new data science or web project, the *very first step* is often installing the required libraries. If a candidate tells you, "The first thing I did was set up a virtual environment and `pip install -r requirements.txt`," they are telling you they are a professional who knows how to properly equip a mission from the start. It's a massive green flag.
