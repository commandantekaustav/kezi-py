Phase 18: The Full Crew Roster

Professor's Briefing

We've learned about lists (for ordered items) and dictionaries (for structured data). Now, we combine them. A **list of dictionaries** is the single most common and powerful way to handle real-world data in Python. This is the master blueprint, the full crew roster, with a complete dossier for every single member.

The Blueprint: A List of Dictionaries

Imagine a list, but instead of each item being a simple string, each item is a complete dictionary (a full dossier).

Code Workbench


# This is a list. It starts with [ and ends with ].
# Each item in the list is a dictionary, starting with { and ending with }.
crew = [
    {
        "name": "Priya Sharma",
        "role": "Data Scientist",
        "score": 92
    },
    {
        "name": "David Chen",
        "role": "ML Engineer",
        "score": 88
    },
    {
        "name": "Kezia Marian",
        "role": "Lead Recruiter",
        "score": 100
    }
]

# We can loop through the list to process each person's dossier.
for member in crew:
    # 'member' is now a dictionary for the current person in the loop.
    name = member["name"]
    role = member["role"]
    score = member["score"]

    print(f"\nAnalyzing {name} ({role})...")

    # Now we can use our 'if' logic on each person's data.
    if score >= 90:
        print("Result: Top-tier asset. APPROVED.")
    else:
        print("Result: Meets standards. APPROVED.")

Professor's Note: Even in a list of exceptional people, your entry, with a score of 100, stands out. Some things are just objective fact.
