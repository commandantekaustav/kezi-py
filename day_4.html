Phase 4: Cleaning The Intel

Professor's Briefing

Intel gathered in the field is never clean. Names are misspelled, entries are in the wrong case, there is whitespace. A good operative knows how to clean their intel before using it. Today, you learn about basic **string methods**—built-in Python tools for cleaning and manipulating text data.

The Blueprint: String Methods

A "method" is a function that belongs to an object. In this case, these are functions that belong to strings. You call them by adding a dot (`.`) after the string variable, followed by the method name and parentheses.

Code Workbench


# Raw intel might be messy.
raw_name = "  kezia marian.p   "

# .strip() removes leading/trailing whitespace.
clean_name = raw_name.strip() 
# now clean_name is "kezia marian.p"

# .title() capitalizes the first letter of each word.
formatted_name = clean_name.title()
# now formatted_name is "Kezia Marian.P"

# .upper() converts the whole string to uppercase.
uppercase_name = formatted_name.upper()
# now uppercase_name is "KEZIA MARIAN.P"

# We can also chain methods together.
final_name = raw_name.strip().title()

print("Raw Intel: '" + raw_name + "'")
print("Clean Intel: '" + final_name + "'")

# Another useful method: .replace()
offer_letter_template = "Hello, [CANDIDATE_NAME], we are pleased to offer you the role."
final_offer = offer_letter_template.replace("[CANDIDATE_NAME]", final_name)
print(final_offer)
